# 🛒 E-Commerce Customer Journey Dashboard | SQL + Power BI

An interactive Business Intelligence dashboard built using SQL and Power BI to analyze customer behavior, revenue trends, traffic sources, product performance, and conversion insights for an e-commerce business.

---

# 📌 Executive Summary

This project focuses on analyzing the complete customer journey of an e-commerce platform — from website sessions and traffic acquisition to orders, revenue generation, refunds, and customer engagement.

Using SQL for data preparation and Power BI for visualization, the dashboard provides business-focused insights into:

- Revenue and order trends
- Customer conversion performance
- Device-based traffic behavior
- Product-level revenue contribution
- Traffic source analysis
- Refund trends and business performance indicators

The goal of this project was to transform raw transactional and session-level data into a visually interactive dashboard that supports strategic business decision-making.

---

# 🎯 Business Problem

E-commerce businesses generate large volumes of customer and transaction data every day. However, without proper analysis, it becomes difficult to:

- Understand customer behavior
- Identify high-performing traffic sources
- Track conversion efficiency
- Analyze revenue growth patterns
- Monitor refunds and customer retention
- Optimize product performance

This dashboard helps stakeholders quickly monitor business performance and identify key operational insights through a centralized analytics solution.

---

# ⚙️ Methodology

The project was completed in the following stages:

### 1️⃣ Data Collection
Imported raw e-commerce datasets including:
- Orders
- Order Items
- Website Sessions
- Website Pageviews
- Products
- Refund Data

### 2️⃣ Data Preparation (SQL)
- Created relational tables in PostgreSQL
- Imported CSV datasets
- Cleaned and validated data
- Built relationships between datasets
- Performed aggregations and business calculations

### 3️⃣ Data Modeling (Power BI)
- Established table relationships
- Created calculated measures using DAX
- Designed KPI metrics and business indicators

### 4️⃣ Dashboard Design
- Built a modern dark-themed dashboard UI
- Added interactive slicers and filters
- Applied business storytelling techniques
- Created visually consistent analytics cards and charts

# 📈 Key Results & Insights

- Website traffic showed strong growth trends over time.
- Mobile and desktop traffic behavior revealed customer engagement patterns.
- Certain traffic sources generated significantly higher sessions and conversions.
- Product-level analysis identified top revenue-generating products.
- Refund rates remained relatively low, indicating healthy customer satisfaction.
- Revenue trends highlighted strong seasonal performance during year-end months.

---

# 💡 Business Recommendations

Based on the analysis, the following recommendations can help improve business performance:

- Invest more in high-performing traffic acquisition channels.
- Optimize low-converting traffic sources.
- Focus marketing campaigns on top-performing products.
- Improve customer retention strategies for repeat purchases.
- Monitor refund trends continuously to maintain customer satisfaction.
- Enhance mobile experience to improve conversion rates.
